# Font Generator Tool

Deployed at: https://rforrupesh.github.io/dynamicblogandtool/

A font generator tool with multiple utilities including fancy text, cursive, bold, and more.

## Deploy

This site is deployed via GitHub Pages. Push to the `main` branch and GitHub Pages will serve from the root of the repository.

> **Note:** The `.nojekyll` file is included so GitHub Pages serves the static HTML files directly without Jekyll processing.
